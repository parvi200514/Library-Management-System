# Installation Guide

## Requirements

- Python 3.x
- MySQL Server
- MySQL Workbench
- VS Code

## Setup Steps

1. Install Python.
2. Install MySQL.
3. Import database schema:
   library_management_system.sql
4. Configure database connection.
5. Run main.py
