# User Manual

## Introduction

The Library Management System is designed to automate library operations such as book management, student management, issue transactions, return transactions, and report generation.

---

## Login Process

1. Launch the application.
2. Enter Username.
3. Enter Password.
4. Click Login.

Default Credentials:

Username: admin

Password: admin123

---

## Add Book

1. Navigate to Book Management.
2. Click Add Book.
3. Enter:
   - Title
   - Author
   - Publisher
   - Category
   - ISBN
   - Quantity
4. Save the record.

---

## Add Student

1. Navigate to Student Management.
2. Click Add Student.
3. Enter student details.
4. Save the record.

---

## Issue Book

1. Select Student.
2. Select Book.
3. Verify availability.
4. Click Issue Book.

---

## Return Book

1. Search transaction.
2. Select issued book.
3. Click Return Book.

---

## Generate Reports

Reports Available:

- Available Books
- Issued Books
- Returned Books
- Student Reports

---

## Logout

Click Logout from Dashboard.
