# Library Management System

## Overview

The Library Management System is a desktop-based application developed using Python and MySQL. The system automates library operations such as managing books, students, issue transactions, return transactions, and report generation.

## Features

* Secure Login System
* Book Management
* Student Management
* Issue Book
* Return Book
* Report Generation

## Technology Stack

* Python
* Tkinter
* MySQL
* GitHub

## Modules

1. Authentication Module
2. Book Management Module
3. Student Management Module
4. Issue Book Module
5. Return Book Module
6. Report Generation Module
